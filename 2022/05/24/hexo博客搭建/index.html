HEXO博客搭建

hexo博客搭建

非常感谢博主Fangzh的搭建教程

博主CSDN搭建连接：https://blog.csdn.net/sinat_37781304/article/details/82729029

搭建步骤

  1. 安装Git

    Git安装

  2. 安装node.js

    node.js安装

  3. 安装Hexo

    创建blog文件夹，输入安装命令：

    npm install -g hexo-cli

    初始化：

    cd myblog //进入这个myblog文件夹

    npm install

  4. 创建github个人仓库

  5. 生成SSH添加到Github中

    回到git bash，输入：

    git config --global user.name "yourname"

    git config --global user.email "youremail"

    其中yourname是github用户名，youremail是github的注册邮箱

    创建SSH，回车：

    ssh-keygen -t rsa -C "youremail"

​ 在命令行收到的路径中找到id_rea.pub，将其添加到SSH Key中

​ 回到git bash,查看ssh -T git@github.com是否成功

  1. 将Hexo部署到github中，点击配置文件_config.yml

    deploy:

    type: git

    repo: https://github.com/YourgithubName/YourgithubName.github.io.git

    branch: master

    安装部署命令：

    npm install hexo-deployer-git --save

    然后：

    hexo clean

    hexo generate

    hexo deploy

    注意：这里在执行deploy时，会要求输入用户名密码，接着fatal错误

    解决方法：

    1. 现在Hexo的yml配置里面deploy的repo要用

      giti@github.com:username/reponame.github.io.git

      而不是https://github.com/username/reponame.github.io.git

    2. git修改ssh和http上传模式

      修改为https:git remote set-url origin https://github.com/yourgithubname/yourgithubname.git

      修改为ssh: git remote set-url origin git@github.com:yourgithubname/yourgithubname.git

      建议使用修改为ssh的方案进行代码上传

    主题更换

    点这里：hexo主题下载

    直接在github链接上下载下来，然后放到theme文件夹下就行了，然后再在刚才说的配置文件中把theme换成那个主题文件夹的名字，它就会自动在theme文件夹中搜索你配置的主题。

    而后进下载的主题这个文件夹，可以看到里面也有一个配置文件_config.xml，貌似它默认是_config.xml.example，把它复制一份，重命名为_config.xml就可以了。这个配置文件是修改你整个主题的配置文件。不是example可以不用更换。

    更换电脑操作

    1. 安装git

    2. 设置git全局邮箱和用户名

      git config –global user.name “yourgithubname”

      git config –global user.email “yourgithubemail”

    3. 设置ssh key

      ssh-keygen -t rsa -C “youremail”

      #生成后填到github和coding上（有coding平台的话）

      #验证是否成功

      ssh -T git@github.com

      ssh -T git@git.coding.net #(有coding平台的话)

    4. 安装node.js

      sudo apt-get install nodejs

      sudo apt-get install npm

    5. 安装Hexo

      sudo npm install hexo-cli -g 不需要初始化

    6. git clone git@………………

    7. 进入克隆文件夹：

      cd xxx.github.io

      npm install

      npm install hexo-deployer-git –save

    8. 生成，部署

      hexo g

      hexo d

    9. 写新博客:

      hexo new newpage

    **Tips:**每次写完最好把源文件上传

    ​ git add .

    ​ git commit –m "xxxx"

    ​ git push

    ​ 如果是在已经编辑过的电脑上，已经有clone文件夹了，那么，每次只要和远端同步一下就行了

​ git pull
